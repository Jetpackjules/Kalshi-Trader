1. SHADOW VS LOCAL ROI (High-Resolution Equity Curve)

Compares Live Equity (Blue), Legacy Backtest (Orange), and Unified Backtest (Purple).

--------------------------------------------------------------------------------
2. UNIFIED COMPARISON (Single Strategy, Bulk Style)
--------------------------------------------------------------------------------
Generates the 2-panel "Equity + Daily Return" graph with ROI bars.
* GRANULARITY: Daily (Resampled from per-tick data).
* Use --only-strategy to run a specific strategy.
* Use --warmup-hours (default 48) for accurate initial state.

Command:
python tools/run_unified_comparison.py --snapshot vm_logs/snapshots/snapshot_2026-01-08_173738.json --out backtest_charts/unified_comparison_single.html --only-strategy hb_notional_010 --warmup-hours 48

--------------------------------------------------------------------------------
3. UNIFIED COMPARISON (Multi-Strategy / Bulk Analysis)
--------------------------------------------------------------------------------
Automates running MULTIPLE backtest variants and plotting them on ONE chart.
* GRANULARITY: Daily (Resampled).
* Runs "Normal Hours" and "All-Day" (24/7) versions for multiple strategies.

Command:
python tools/run_unified_comparison.py --snapshot vm_logs/snapshots/snapshot_2026-01-08_173738.json --out backtest_charts/unified_comparison_variants.html --preset full --warmup-hours 48

--------------------------------------------------------------------------------
4. SHADOW VS LOCAL CASH (Cash Balance)
