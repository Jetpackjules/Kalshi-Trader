GRAPH GENERATION INSTRUCTIONS

This file contains the commands to generate all 4 types of backtest analysis graphs.
Ensure you are in the root directory: c:\Users\jetpa\OneDrive - UW\Google_Grav_Onedrive\kalshi_weather_data

--------------------------------------------------------------------------------
1. SHADOW VS LOCAL ROI (Equity Curve)
--------------------------------------------------------------------------------
Compares the live system's equity (from logs) vs the backtest's equity.
* Use --skip-backtest if you only want to re-plot existing data.
* Use --start-ts to align the graph start time.

Command:
python tools/generate_live_vs_backtest_graph.py --snapshot vm_logs/snapshots/snapshot_2026-01-08_173738.json --out-dir unified_engine_out_snapshot_live_v2 --skip-backtest --graph backtest_charts/shadow_vs_local_roi_timeline.html --start-ts "2026-01-08 17:37:38"

--------------------------------------------------------------------------------
2. SHADOW VS LOCAL CASH (Cash Balance)
--------------------------------------------------------------------------------
Compares the raw cash balance over time. Useful for seeing payouts and spending.

Command:
python tools/generate_live_vs_backtest_cash_graph.py --snapshot vm_logs/snapshots/snapshot_2026-01-08_173738.json --out-dir unified_engine_out_snapshot_live_v2 --graph backtest_charts/shadow_vs_local_cash_timeline.html --start-ts "2026-01-08 17:37:38"

--------------------------------------------------------------------------------
3. UNIFIED VARIANT (Single Strategy, Legacy Style)
--------------------------------------------------------------------------------
Generates the 2-panel "Equity + Daily Return" graph for a SINGLE backtest run.
Matches the style of the legacy ComplexBacktester.

Prerequisite: Run a backtest first (e.g., python run_unified_backtest.py ...)

Command:
python tools/generate_unified_variant_graph.py --snapshot dec05_snapshot.json --out-dir unified_engine_out_dec05 --out backtest_charts/unified_variant_dec05.html

--------------------------------------------------------------------------------
4. UNIFIED COMPARISON (Multi-Strategy, Hours vs All-Day)
--------------------------------------------------------------------------------
Automates running MULTIPLE backtest variants and plotting them on ONE chart.
This is the "Master" command for strategy analysis.
* Runs "Normal Hours" and "All-Day" (24/7) versions for multiple strategies.
* Generates the comparison graph automatically.

Command:
python tools/run_unified_comparison.py --snapshot dec05_snapshot.json --start-ts "2025-12-05 00:00:00" --initial-cash 100.0 --out backtest_charts/unified_comparison_dec05.html

--------------------------------------------------------------------------------
HELPER: RUNNING A SINGLE BACKTEST
--------------------------------------------------------------------------------
If you need to generate data for Graph #3 manually:

Command:
python run_unified_backtest.py --snapshot dec05_snapshot.json --start-ts "2025-12-05 00:00:00" --initial-cash 100.0 --warmup-hours 0 --out-dir unified_engine_out_dec05 --strategy hb_notional_010
